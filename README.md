# 📊 Job Salary & Career Insights Analysis

## 🔍 Project Overview

This project analyzes job salary trends using SQL and Power BI.
It helps understand how experience, skills, education, and industry impact salaries.

---

## 🧰 Tools Used

* SQL (Data Analysis)
* Power BI (Data Visualization)
* Excel / CSV Dataset

---

## 📁 Dataset

The dataset includes:

* Job Title
* Experience Years
* Education Level
* Skills Count
* Industry
* Company Size
* Location
* Remote Work
* Certifications
* Salary

---

## 📊 Key Analysis

* Salary vs Experience
* Skills vs Salary
* Industry-wise Salary
* Remote vs Non-Remote Jobs
* Certifications Impact

---

## 📈 Power BI Dashboard

Created interactive dashboards with:

* KPI Cards (Avg Salary, Total Jobs)
* Bar Charts & Line Charts
* Filters (Slicers)

---

## 💡 Key Insights

* Experience increases salary significantly
* More skills lead to higher salary
* Certifications improve earning potential
* Remote jobs offer competitive salaries

---

## 📎 Project Files

* `job_prediction.csv` → Dataset
* `queries.sql` → SQL Queries
* `dashboard.pbix` → Power BI File
* `project_report.pdf` → Project Report

---

## 🚀 How to Use

1. Load dataset into SQL
2. Run queries from `queries.sql`
3. Open Power BI file to view dashboard
